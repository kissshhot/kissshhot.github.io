回溯模板

核心思路

  • 全排列：used 数组记录已选，每次从未选中的数中选一个加入 path。
  • 子集：从当前 index 开始枚举，每个节点都可以收录到结果集中。
  • 组合总和：允许重复选，每次从当前 index 开始；剩余 target < 0 则剪枝。
  • 括号生成：剩余左括号数 > 0 可加 (；剩余右括号数 > 剩余左括号数可加 )。
  • N 皇后：用三个集合分别记录已占用的列、左对角线、右对角线，逐行放置皇后。
